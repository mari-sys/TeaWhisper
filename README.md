# TeaWhisper
## サービス概要
TeaWhisperは、お茶の香りや心が整う瞬間を写真とともに共有できるコミュニティアプリです。
紅茶・日本茶・ハーブティーなど幅広いお茶文化を対象に、日常の小さな癒しを穏やかに記録・発信できます。
SNSのように"見られること"を意識するのではなく、自分のペースでお茶時間を味わうことを大切にしています。

## このサービスへの思い・作りたい理由
仕事や勉強の合間にお茶を淹れる時間は、私にとって心を整える大切な習慣でした。
しかし、既存のSNSではコーヒー文化が中心で、紅茶や日本茶のような“静かな癒やし”の投稿は埋もれてしまいがちです。

「もっと穏やかに、自分の好きなお茶を語れる場所があったらいいのに」——そんな思いから、このサービスを企画しました。

TeaWhisperは、人と比べない・競わない・穏やかに繋がるSNSです。
お気に入りの茶葉を記録したり、心に響く香りを共有したり。お茶を通して、自分と向き合う時間を後押しします。

## ユーザー層について

- お茶・紅茶が好きな方
  → SNSで日常の癒しを共有する文化がある。
- カフェ巡り・喫茶店好き
  → 紅茶・和カフェ情報が少なく、共感できる仲間を探している。
- ハーブティー・健康志向層
  → 心身を整える時間を大切にしており、落ち着いたSNSを求めている。


## サービスの利用イメージ
- 「今日のお茶」投稿
 自宅で淹れたお茶の写真に、香りや気分をタグで記録
→ 自分だけの癒やしアルバムとして残る

-  カフェ探索モード（本リリース予定）
地図上から紅茶カフェや投稿を探す
→ 「行きたいお茶カフェリスト」を作成できる

-  My Tea Shelf
飲んだお茶をコレクション化して、自分の“香りの履歴書”を作る


## ユーザーの獲得について
XやInstagramなどのSNSでの共感投稿を起点に、
紅茶やお茶文化に関心をもつユーザーに自然に広がる導線を設計します。

## サービスの差別化ポイント・推しポイント

### MVPでも実現する差別化
- **お茶文化に特化**: 紅茶・緑茶・ハーブティーなど全てのお茶を歓迎
- **穏やかなコミュニティ設計**: 応援マーク（数値非表示）で静かに共感を伝える
- **自分のペースで記録**: "見られること"を意識せず、癒しの時間を大切に

### 本リリースで追加予定の差別化
- 香りや気分をタグで表現（#ほっとする、感覚的な検索）
- お茶カフェを地図で探索（Google Maps API）
- My Tea Shelfでコレクション化（香りの履歴書）


## 機能候補

##  MVP機能一覧

### 1. ユーザー機能
- ユーザー登録・ログイン（Devise）
- 簡単なプロフィール（名前・自己紹介）

### 2. 投稿機能
- お茶投稿の作成・編集・削除
  - 写真アップロード
  - お茶の種類（選択式）
  - ひとことメモ
- 投稿一覧（新着順）
- 投稿詳細ページ

### 3. 交流機能
- 応援マーク機能（数値は非表示）
- マイページ（自分の投稿一覧）

### 4. 基本機能
- レスポンシブデザイン対応
- 画像アップロード機能
- デプロイ（Render × Docker）対応
